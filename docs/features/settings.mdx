# 설정 및 계정 관리

설정 페이지는 개인화된 학습 환경을 구성하고 계정을 관리할 수 있는 기능입니다.

알림 설정, 에디터 환경, 개인화 옵션 등을 사용자의 학습 스타일과 선호도에 맞게 조정할 수 있어요.

## 설정 페이지 접근

### 설정 메뉴 찾기
- **사이드바**: 좌측 네비게이션 메뉴의 "설정" 클릭
- **프로필 메뉴**: 우상단 프로필 이미지 → "계정 설정"
- **단축키**: `Ctrl/Cmd + ,` 키로 빠른 접근

### 설정 카테고리 구성
```
설정 페이지 구조:
├── 계정 관리
├── 알림 설정
├── 학습 환경
├── 개인화 설정
├── 연동 서비스
├── 데이터 관리
└── 고급 설정
```

## 계정 관리

### 기본 프로필 설정

**개인 정보**
- **닉네임**: 커뮤니티에서 표시될 이름
- **이메일**: 로그인 및 알림 수신용 주 이메일
- **프로필 이미지**: 개인 사진이나 아바타 설정
- **자기소개**: 간단한 소개글 (선택사항)

**학습 프로필**
```
예시 학습 프로필:
학습 분야: 웹 개발, 데이터 분석
현재 목표: 풀스택 개발자 되기
학습 스타일: 체계적, 실습 중심
선호 시간: 오전 (7-9시), 저녁 (8-10시)
```

### 보안 설정

**비밀번호 관리**
- 현재 비밀번호 변경
- 보안 강도 확인 (약함/보통/강함)
- 정기 비밀번호 변경 알림 설정

**2단계 인증**
- Google Authenticator 연동
- SMS 인증 설정
- 백업 코드 생성 및 다운로드
- 로그인 기록 및 의심스러운 활동 모니터링

**계정 연동**
- Google 계정 연동
- GitHub 계정 연동
- 다른 소셜 로그인 추가/제거

## 알림 설정

### 복습 알림 커스터마이징

**알림 시간 설정**
```
개인별 최적 알림 시간 예시:

직장인 패턴:
- 오전 출근 전: 07:30
- 점심시간: 12:30  
- 퇴근 후: 19:00

학생 패턴:
- 등교 전: 08:00
- 쉬는 시간: 15:30
- 저녁 시간: 20:00
```

**알림 빈도 조절**
- **집중 모드**: 하루 1회만 알림
- **표준 모드**: 하루 2-3회 알림
- **적극 모드**: 학습 기회가 있을 때마다 알림
- **수동 모드**: 사용자가 직접 접속할 때만

**망각곡선 기반 설정**
- 새 내용: 1일, 3일, 7일, 14일, 30일 후
- 어려운 내용: 더 짧은 간격으로 반복
- 쉬운 내용: 더 긴 간격으로 복습
- 개인 정답률에 따른 자동 조절

### 커뮤니티 알림

**상호작용 알림**
- 내 글에 댓글이 달렸을 때
- 내 글이 좋아요를 받았을 때  
- 누군가 나를 팔로우했을 때
- 멘션(@닉네임)을 받았을 때

**콘텐츠 알림**
- 팔로우하는 사용자의 새 글
- 관심 태그의 새 글
- 인기 글 주간/월간 요약
- 내가 참여한 토론의 새 댓글

## 학습 환경 설정

### 에디터 커스터마이징

**글쓰기 환경**
- **테마**: 라이트/다크/자동 (시스템 설정 따름)
- **폰트**: 본문용 (나눔고딕/맑은고딕/Custom), 코드용 (Monaco/Fira Code)
- **폰트 크기**: 소/보통/대/특대
- **줄 간격**: 좁게/보통/넓게

**에디터 기능**
- **자동저장 간격**: 10초/30초/1분/5분
- **맞춤법 검사**: 한국어/영어/비활성화
- **슬래시 메뉴 활성화**: 빠른 블록 추가 메뉴
- **마크다운 단축키**: 빠른 서식 적용

**미리보기 설정**
- **실시간 미리보기**: 작성과 동시에 결과 확인
- **분할 화면**: 에디터와 미리보기 나란히 보기
- **전체 화면**: 방해 요소 없는 집중 모드

### 대시보드 레이아웃

**위젯 배치**
```
대시보드 커스터마이징 예시:

집중형 배치 (개발자용):
├── 학습 통계 (상단 전체)
├── 최근 프로젝트 (좌측 상단)
├── 코딩 시간 추적 (우측 상단)
└── 기술 스택 성장 (하단)

균형형 배치 (일반 학습자):
├── 오늘의 복습 (좌측)
├── 학습 달력 (중앙)
└── 커뮤니티 피드 (우측)
```

**정보 밀도 조절**
- **간단히**: 핵심 정보만 표시
- **표준**: 기본 정보량
- **상세히**: 모든 세부 정보 표시

## 개인화 설정

### AI 복습 튜닝

**질문 생성 스타일**
- **소크라테스식**: 답을 유도하는 질문 중심
- **직접적**: 명확한 답변을 요구하는 질문
- **응용형**: 실제 상황 적용 문제 중심
- **혼합형**: 다양한 스타일 조합

**난이도 조절**
- **현재 수준 유지**: 기존 이해도 기준
- **점진적 향상**: 조금씩 어려운 문제 제시
- **도전적**: 현재 수준보다 높은 문제
- **자동 조절**: AI가 성취도에 따라 자동 조절

### 학습 목표 관리

**목표 설정**
```
SMART 목표 설정 예시:

목표: React 하기
├── Specific: React Hook, Context API, 성능 최적화 이해
├── Measurable: 실제 프로젝트 3개 완성
├── Achievable: 하루 2시간씩 꾸준히 학습
├── Relevant: 프론트엔드 개발자 취업 준비
└── Time-bound: 3개월 내 완료
```

**진행률 추적**
- 주간/월간 목표 대비 실제 성과
- 학습 시간 vs 계획 시간 비교
- 완료한 마일스톤 시각화
- 목표 달성률 예측 및 조언

## 연동 서비스

### 외부 플랫폼 연동

**개발자 도구 연동**
- **GitHub**: 프로젝트 진행상황과 코드 연동
- **Notion**: 기존 노트와 동기화
- **Obsidian**: 개인 지식 관리 도구 연결
- **Anki**: 스페이스드 리피티션 카드 동기화

**생산성 도구**
- **Google Calendar**: 학습 일정 자동 등록
- **Todoist**: 학습 관련 할 일 동기화
- **RescueTime**: 실제 학습 시간 자동 추적
- **Forest**: 집중시간 관리 연동

### 데이터 내보내기

**백업 생성**
- **전체 데이터**: 모든 글, 댓글, 설정 포함
- **글만**: 작성한 모든 학습 기록
- **통계만**: 학습 패턴과 성과 지표
- **선택적**: 원하는 데이터만 골라서

**내보내기 형식**
```
지원 포맷:
- JSON: 개발자용 구조화된 데이터
- Markdown: 다른 플랫폼 호환용
- PDF: 출력 및 보관용
- CSV: 통계 분석용
- HTML: 웹 브라우저용
```

## 데이터 관리

### 저장소 관리

**용량 사용량 확인**
- 전체 사용량 vs 할당량
- 파일 유형별 사용량 (이미지/동영상/문서)
- 월별 사용량 추이
- 저장공간 최적화 제안

**정리 도구**
- 중복 파일 찾기 및 삭제
- 사용하지 않는 이미지 정리
- 오래된 임시저장 파일 삭제
- 대용량 파일 압축 제안

### 개인정보 보호

**데이터 공개 범위**
```
공개 수준 설정:
├── 전체 공개: 누구나 볼 수 있음
├── 회원 공개: 가입자만 볼 수 있음
├── 친구 공개: 팔로우 관계에서만
└── 비공개: 본인만 볼 수 있음
```

**익명화 옵션**
- 프로필 정보 숨김 모드
- 통계 참여 시 개인 식별 정보 제외
- 커뮤니티 활동 익명 참여 옵션

## 고급 설정

### 개발자 옵션

**API 설정**
- Personal Access Token 생성
- 웹훅 URL 설정 (새 글 알림 등)
- RSS 피드 커스터마이징
- 자동화 도구 연동 설정

**실험적 기능**
- 베타 기능 미리 체험
- 새 AI 모델 테스트 참여
- UI/UX 개선 피드백 제공
- A/B 테스트 참여 설정

### 접근성 설정

**시각적 접근성**
- 고대비 모드 활성화
- 폰트 크기 확대 (최대 200%)
- 색상 구분 대신 패턴 사용
- 화면 읽기 도구 최적화

**키보드 네비게이션**
- Tab 네비게이션 순서 최적화
- 사용자 정의 단축키 설정
- 마우스 없이 모든 기능 접근
- 음성 명령 지원 (실험적)

## 설정 백업과 복원

### 설정 프로필

**프로필 저장**
- 현재 모든 설정을 하나의 프로필로 저장
- 여러 프로필 생성 (집중모드/휴식모드 등)
- 프로필 간 빠른 전환
- 시간대별 자동 프로필 전환

**설정 공유**
- 다른 사용자와 설정 공유 (선택적)
- 추천 설정 템플릿 적용
- 커뮤니티 인기 설정 확인

### 계정 이전

**기기 간 동기화**
- 모바일 앱과 설정 동기화
- 여러 브라우저 간 설정 공유
- 오프라인 설정 캐시

**계정 마이그레이션**
- 다른 이메일로 계정 이전
- 여러 계정 통합 도구
- 계정 삭제 전 데이터 백업

## 문제 해결

### 일반적 설정 문제

**알림이 안 와요**
1. 브라우저 알림 권한 확인
2. 이메일 스팸 폴더 확인
3. 알림 시간대 설정 재확인
4. 모바일 푸시 알림 권한 확인

**설정이 저장 안 돼요**
1. 브라우저 쿠키 설정 확인
2. 애드블로커 비활성화 시도
3. 다른 브라우저에서 테스트
4. 캐시 및 쿠키 삭제 후 재로그인

**동기화 문제**
1. 인터넷 연결 상태 확인
2. 계정 연동 상태 재확인
3. 수동 동기화 실행
4. 고객지원팀 문의
