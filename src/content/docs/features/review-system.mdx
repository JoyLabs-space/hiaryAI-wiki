hiaryAI의 복습 시스템은 망각곡선 이론에 기반해서 개인 맞춤 복습 일정을 제공하는 기능입니다.

작성한 학습 기록을 분석해서 적절한 시점에 소크라테스식 질문을 생성하고, 개인의 이해도에 따라 복습 주기를 조정해요.

## 복습 시스템의 과학적 원리

### 망각곡선 이론 
독일의 심리학자 헤르만 에빙하우스가 발견한 망각곡선에 따르면, 사람은 학습 직후부터 빠르게 내용을 잊어버려요.

**망각 진행 과정:**
- 20분 후: 42% 기억
- 1시간 후: 56% 기억 
- 1일 후: 74% 기억
- 1주일 후: 77% 기억
- 1개월 후: 79% 기억

### 간격 반복의 효과
하지만 적절한 시점에 복습을 하면 기억을 강화할 수 있어요. hiaryAI는 이 원리를 바탕으로 개인별 최적 복습 주기를 계산해요.

**과학적 복습 주기:**
1. **첫 복습**: 학습 후 1일
2. **두 번째 복습**: 학습 후 3일
3. **세 번째 복습**: 학습 후 7일 
4. **네 번째 복습**: 학습 후 2주
5. **다섯 번째 복습**: 학습 후 1개월

## AI 생성 복습 질문 (개발 예정)

### 소크라테스식 질문법
hiaryAI의 AI는 단순한 암기 확인이 아닌, 깊이 있는 이해를 돕는 소크라테스식 질문을 생성해요.

**질문 유형별 예시:**

**개념 이해 확인:**
- "이 개념을 다른 말로 설명한다면?"
- "이 원리가 중요한 이유는 무엇인가요?"
- "실생활에서 이 개념이 사용되는 예시는?"

**비교와 대조:**
- "A 방법과 B 방법의 차이점은?"
- "이전에 학습한 X 개념과 어떤 연관성이 있나요?"
- "언제 이 방법을 사용하고, 언제 사용하지 않나요?"

**적용과 응용:**
- "이 지식을 실제 프로젝트에 어떻게 적용하시겠어요?"
- "만약 다른 상황이라면 어떻게 접근하시겠어요?"
- "이 개념을 확장하면 어떤 가능성이 있을까요?"

### 구독 플랜별 질문 차이

**무료 플랜:**
기본적인 알림만 제공 (AI 질문 없음)

**스탠다드 플랜 - 기본 AI 질문:**
```
질문: "useState Hook의 주요 특징을 설명해보세요."
```

**프로 플랜 - 집중/맞춤형 AI 질문:**
```
질문: "useState Hook을 사용할 때와 useReducer를 사용할 때를 
어떻게 구분하시겠어요? 각각의 장단점을 구체적인 예시와 
함께 설명해보세요."
```

## 모바일 앱 복습 알림

### 스마트한 알림 시스템

**개인 맞춤형 알림 시간:**
AI가 사용자의 활동 패턴을 학습해서 적절한 알림 시간을 찾아드려요.

- 주로 아침에 활동적이면 → 오전 시간대 알림
- 저녁에 집중도가 높으면 → 저녁 시간대 알림 
- 점심시간을 활용하면 → 점심 시간대 알림

**컨텍스트 기반 알림 내용:**
일반적인 "복습하세요" 알림이 아니라, 학습 내용의 핵심 키워드를 포함한 알림을 보내요.

```
일반적인 알림: "복습할 시간입니다!"

 hiaryAI 알림: "React useState: 상태 업데이트가 
비동기인 이유를 기억하시나요? "
```

### 마이크로 학습 지원

**알림에서 바로 복습:**
알림을 터치하면 전체 글을 읽지 않고도 핵심 내용을 빠르게 복습할 수 있어요.

**진행률 표시:**
이번 복습이 몇 번째인지, 다음 복습은 언제인지 한눈에 확인할 수 있어요.

## 사회적 학습 기능

### 다른 학습자와 함께 복습

**답변 공유 시스템:**
같은 질문에 답변한 다른 학습자들의 답변을 확인할 수 있어요.

```
내 답변: "useState는 함수형 컴포넌트에서 상태를 관리하는 Hook이다."

다른 학습자의 답변: "useState는 상태값과 setter 함수를 배열로 
반환하며, setter 함수 호출 시 컴포넌트가 리렌더링된다. 
중요한 점은 상태 업데이트가 비동기로 처리된다는 것이다."
```

**학습 공동체 형성:**
비슷한 주제를 학습하는 사람들과 자연스럽게 연결되어 함께 성장할 수 있어요.

### 댓글과 토론

**질문에 대한 토론:**
복습 질문을 통해 시작된 토론에서 더 깊이 있는 학습이 일어나요. *(개발 예정)*

**실시간 번역:**
전 세계 학습자들과 언어 장벽 없이 소통할 수 있어요.

## 개인화된 복습 경험

### 학습 패턴 분석

**개인별 망각 주기 측정:**
AI가 사용자의 복습 성과를 분석해서 개인별 최적 주기를 찾아요.

- 빠르게 잊는 분야 → 더 짧은 주기로 복습
- 잘 기억하는 분야 → 더 긴 주기로 복습

**난이도별 조정:**
복습 질문의 난이도도 사용자의 실력에 맞춰 조정돼요. *(개발 예정)*

### 목표 기반 복습

**학습 목표와 연동:**
설정한 학습 목표에 따라 복습 우선순위가 결정돼요.

예: "3개월 내 React 하기" 목표
→ React 관련 복습을 더 자주, 더 집중적으로

## 복습 효과 측정

### 복습 성과 추적

**이해도 변화 측정:**
복습을 통해 개념 이해도가 어떻게 변화하는지 시각적으로 확인할 수 있어요.

**기억 강화 지수:**
과학적 알고리즘을 통해 각 개념의 기억 강화 정도를 수치화해요.

### 복습 성취 시스템

**복습 스트릭:**
연속으로 복습을 완료한 일수를 추적하고 동기부여를 제공해요.

**주제별 마스터리:**
특정 주제에 대한 복습을 충분히 완료하면 "마스터" 배지를 획득해요.

## 구체적인 활용 시나리오

### 개발자의 하루

**아침 (09:00):**
```
 알림: "JavaScript 클로저: 렉시컬 스코프와의 관계를 
설명할 수 있나요?"

💭 복습: 3분간 개념 정리 → 다른 개발자 답변 확인 → 
부족한 부분 메모
```

**점심시간 (12:30):**
```
 알림: "React useEffect: dependency array가 빈 배열일 때와 
없을 때의 차이점은?"

💭 복습: 실제 코드 예시로 생각해보기 → 댓글로 질문
```

### 언어 학습자의 복습

**출근길 (08:00):**
```
 알림: "과거완료시제: had + p.p 형태는 언제 사용하나요?"

💭 복습: 예문 만들어보기 → 원어민 학습자의 답변 확인
```

**저녁 (21:00):**
```
 알림: "관계대명사 which: 제한적 용법과 비제한적 용법의 
차이점을 예문으로 설명해보세요."

💭 복습: 각각의 예문 작성 → 문법 포인트 재확인
```

## 복습 시스템 최적화 팁

### 효과적인 복습 습관

**1. 일정한 시간에 복습하기**
매일 같은 시간에 복습하면 습관이 되어 더 쉬워져요.

**2. 짧고 집중적으로**
한 번에 오래 하기보다는 5-10분씩 집중해서 복습하세요.

**3. 능동적으로 답변하기**
머릿속으로만 생각하지 말고 실제로 답변을 작성해보세요.

**4. 다른 학습자와 비교하기**
다양한 관점의 답변을 보고 자신의 이해도를 점검하세요.

### 분야별 복습 전략

**개발 분야:**
- 코드 예시와 함께 개념 설명
- 실제 프로젝트 적용 사례 생각해보기
- 다른 기술과의 비교·대조

**언어 학습:**
- 새로운 예문 만들어보기 
- 일상 상황에 적용해보기
- 발음이나 억양도 함께 연습

**학술 분야:**
- 핵심 개념의 논리적 연결
- 실제 사례나 현상에의 적용
- 비판적 사고를 통한 한계점 분석

## 자주 묻는 질문

### Q. 복습 알림이 너무 많아요. 조절할 수 있나요?
A. 네, 설정에서 복습 빈도를 조절하거나 특정 시간대만 알림을 받도록 설정할 수 있어요.

### Q. 복습 질문이 너무 어려워요. (개발 예정)
A. AI가 사용자의 답변 패턴을 학습해서 점진적으로 난이도를 조정해요. 처음에는 어려울 수 있지만 꾸준히 하시면 적절한 수준으로 맞춰져요.

### Q. 복습을 놓쳤을 때는 어떻게 되나요?
A. 놓친 복습은 다음 날 다시 알림이 와요. 하지만 너무 많이 밀리지 않도록 꾸준히 하시는 게 좋아요.

### Q. 오프라인에서도 복습할 수 있나요?
A. 모바일 앱에서 최근 복습 내용을 오프라인으로 저장해서 인터넷 없이도 복습할 수 있어요.
