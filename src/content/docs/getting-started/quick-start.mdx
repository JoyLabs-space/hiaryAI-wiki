hiaryAI를 처음 사용할 때 필요한 기본 설정과 첫 번째 학습 기록 작성 방법을 안내합니다.

이 가이드를 따라하면 계정 생성부터 첫 글 작성, AI 복습 기능 체험까지 단계별로 진행할 수 있어요.

## 목표

이 가이드를 완료하면 다음을 할 수 있게 돼요:
- hiaryAI 계정을 만들고 기본 설정을 완료해요
- 학습 에디터로 첫 번째 학습 기록을 작성해요
- AI가 생성한 복습 질문을 체험해봐요
- 모바일 앱으로 복습 알림을 받을 수 있게 설정해요 *(개발 예정)*

## 사전 요구사항

- 웹 브라우저 (Chrome, Safari, Firefox 등)
- 이메일 주소
- 모바일 기기 (선택사항, 복습 알림용)

## 1단계: 계정 만들기

### 회원가입 및 온보딩
1. **초기 화면 접근**
   - [hiaryAI 웹사이트](https://hiary.ai)에 접속하세요
   - 깔끔한 로그인 화면이 표시됩니다

🔐 **[이미지 플레이스홀더: hiaryAI 메인 로그인 화면]**

2. **회원가입 선택**
   - "회원가입" 버튼을 클릭하세요
   - 이메일과 비밀번호를 입력하여 계정을 생성하세요

3. **소셜 로그인 옵션**
   - Google, GitHub 등 소셜 계정으로 간편 가입 가능
   - 별도 비밀번호 설정 없이 바로 시작할 수 있습니다

### 온보딩 프로세스
hiaryAI는 단계별 온보딩을 통해 개인 맞춤 설정을 도와요:

🚀 **[이미지 플레이스홀더: 온보딩 프로세스 4단계]**

**1단계 - 기본 프로필**
- **닉네임**: 커뮤니티에서 사용할 이름을 입력하세요
- **프로필 이미지**: 선택사항이지만 커뮤니티 활동에 도움이 됩니다

**2단계 - 학습 분야 선택**
- 주요 관심 분야를 선택하세요:
  - 개발/프로그래밍
  - 언어 학습  
  - 디자인
  - 학술 연구
  - 자격증 준비
  - 기타

**3단계 - 학습 목표 설정**
- 구체적인 학습 목표를 작성해보세요
- 예: "3개월 내 React 하기", "토익 900점 달성"

**4단계 - 알림 설정**
- 복습 알림을 받을 시간대를 설정합니다
- 개인의 생활 패턴에 맞는 시간을 선택하세요

## 2단계: 첫 학습 기록 작성하기

### 대시보드 둘러보기
온보딩을 마치면 hiaryAI의 메인 대시보드가 나타납니다:

🏠 **[이미지 플레이스홀더: hiaryAI 메인 대시보드]**

- **좌측 사이드바**: 네비게이션 메뉴 (홈, 내 글, 탐색, 설정 등)
- **중앙 영역**: 메인 피드와 학습 현황

### 새 글 쓰기 시작
1. **글 작성 버튼 찾기**
   - 대시보드 상단의 "새 글 쓰기" 버튼을 클릭하세요
   - 또는 사이드바의 "+" 아이콘을 사용할 수 있습니다

2. **제목 작성**
   - 구체적이고 명확한 제목을 입력하세요
   - 좋은 예: "React Hook useState ", "영어 과거완료 시제 정리"
   - 피할 예: "오늘 공부", "React 공부"

### 블록형 에디터 사용하기
hiaryAI의 에디터는 다양한 블록을 지원해요:

📝 **[이미지 플레이스홀더: 블록형 에디터 화면]**

**텍스트 블록:**
- 일반 텍스트로 학습한 내용을 정리하세요
- **볼드체**나 *이탤릭체*로 중요한 부분을 강조하세요

**코드 블록 (개발자용):**
```javascript
// 슬래시(/) 메뉴에서 '코드'를 선택하세요
const [count, setCount] = useState(0);
```

**수식 블록 (수학/과학 학습자용):**
- LaTeX 문법으로 수식을 입력하세요
- 예: `$E = mc^2$` 또는 `$$\sum_{i=1}^{n} x_i$$` 형태로 작성



### 발행하기
1. '발행하기' 버튼을 클릭하세요
2. 최종 발행 확인!

## 3단계: AI 복습 기능 체험하기

### 복습 질문 확인하기 (개발 예정)
1. 작성한 글 페이지에서 '복습 질문' 섹션을 찾으세요
2. AI가 생성한 소크라테스식 질문들을 확인하세요 *(개발 예정)*

**예시 질문들 (개발 예정):**
- "이 개념을 다른 예시로 설명한다면 어떻게 하시겠어요?"
- "이 방법의 장단점은 무엇인가요?"
- "비슷한 개념과 어떤 차이가 있을까요?"

### 질문에 답변하기
1. 각 질문에 대한 답변을 작성해보세요
2. 답변 제출 후 다른 학습자들의 답변도 확인해보세요
3. 좋은 답변에는 '좋아요'나 댓글로 소통하세요

## 4단계: 모바일 앱 설정하기 (개발 예정)

### 복습 알림 설정
1. 앱의 '설정' → '알림' 메뉴로 이동
2. 복습 알림을 켜고 원하는 시간대를 설정하세요
3. 알림 권한을 허용하세요

**추천 알림 시간:**
- 오전: 하루 계획 세울 때
- 오후: 점심시간이나 잠깐의 휴식시간
- 저녁: 하루 마무리하며 복습할 때

## 5단계: 첫 복습 경험하기

### 복습 알림 받기
- 첫 학습 기록 작성 후 1일 뒤에 첫 복습 알림이 와요
- 알림을 터치해서 복습 페이지로 이동하세요

### 복습하기
1. AI가 생성한 질문에 다시 답변해보세요
2. 이전 답변과 비교해서 이해도를 체크하세요
3. 잘 기억나지 않는 부분은 원본 글을 다시 읽어보세요
